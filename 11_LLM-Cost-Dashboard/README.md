# LLM Usage & Cost Monitoring Dashboard

Governance-focused system for tracking LLM usage, token consumption, cost allocation, and executive reporting.

## Objectives

- Track token usage by model
- Convert token usage to cost
- Allocate costs by department
- Monitor budget thresholds
- Provide executive-level summary metrics

## Governance Controls

- Cost threshold alerts
- Departmental usage caps
- Model-level cost tracking
- Monthly reporting framework

## Structure

- data/ — raw usage exports
- dashboard/ — Excel or Power BI files
- docs/ — definitions and formulas
- case-study/ — executive case study
